You found this file by running
	ls -a
the a is what we call a flag it modifies the behavior of the command we're running
adding -a in this case causes ls to display ALL files including hidden ones
you may have noticed the files . and .. these are special files that exist in every directory
. refers to this directory
.. refers to the directory above this one.

Go ahead and cd to the directory above this one by typing
	cd ..
after doing that cd into the next directory with
	cd paths
